README.md
1 The MSP430 SNES Controller Library
2 ==================================
3 
4 Overview
5 --------
6 
7 This library purpose is to enable the operation of one or several SNES
8 controllers with a MSP430 microcontroller. It handles the communication
9 protocol with the controller's shift register, returning a structure with all
10 buttons states.
11 
12 To create end initialize a controller, use the snesCtrlSetup() to create a
13 SnesCtrl instance. Each gamepad is represented by a single SnesCtrl instance.
14 The snesCtrlRead() function read the controler's state, from the SnesCtrl
15 instance, and returns a SnesCtrlBtnPressed structure instance with all
16 buttons states.
17 
18 Installation
19 ------------
20 The installation process is as simple as coping both files (snes_ctrl.h and snes_ctrl.c) inside your project and including the .h header where the routines will be used.
21 
22 Build Instructions
23 ------------------
24 
25 The library should be build statically whitin your project.
26 
27 It's intent is to be build with the TI MSP430 compiler, if you want to use the msp430gcc, it is necessary to replace the DELAY_12us and DELAY_6us macros inside the snes_ctrl.h file to some function/macro that will skip the right amount of time.
28 
29 Electrical Connections
30 ----------------------
31 
32 The following diagram show how to wire the SNES controller to the MSP430:
33 
34  MSP430xxxx
35  -----------------
36  /|\| PORT.DATA|<--10kR<--CONTROLLER DATA PIN
37  | | |
38  --|RST PORT.CLOCK|-->10kR-->CONTROLLER CLOCK PIN
39  | |
40  | PORT.LATCH|-->10kR-->CONTROLLER LATCH PIN
41  ...................
42 
43 NOTES:
44 
45  * The port can be 1 or 2, but it must be the same for all pins.
46  * DATA, CLOCK and LATCH can be any pins.
47  * The 10k Ohms resistor is needed to limit the current, since the SNES controller operates at 5V.

49 Usage Example
50 -------------
51 
52 The code was compiled with the TI MSP430 compiler and successfully tested
53 with a MSP430G2452 microcontroller.
54 
55  #include "snes_ctrl.h"
56 
57  // Set clock speed to 2Mhz (this is only necessary if the clock frequency is
58  // not 1Mhz).
59  #define _MCU_CLOCK_ 2000000
60 
61  int main(){
62  // Create the controller configuration and initialize peripherals.
63  // The controller is hooked to pins P2.2 (data), P2.0 (clock) and P2.1
64  // (latch).
65  SnesCtrl ctrl = snesCtrlSetup(2, 2, 0, 1);
66 
67  // Create the button structure to receive the buttons state.
68  SnesCtrlBtnPressed btn;
69 
70  // Read the buttons state.
71  btn = snesCtrlRead(&ctrl);
72 
73  // Do something if button B is pressed.
74  if(btn.b)
75  something();
76 
77  // infinite loop to hold the program.
78  while(1);
79  }
